HTML5

Hypertext Markup Language (HTML) es un lenguaje de marcado del lado del cliente desarrollado por Tim Berners-Lee que surgió a principios de los 90's. Es un idioma que hace uso de etiquetas para crear la estructura de las páginas web; este se complementa con otras tecnologías y lenguajes para el maquetado web (crear sitios web visualmente atractivos y dinámicos).

Durante sus primeros años solo contaba con 18 etiquetas, pero, con el pasar de los años, se fueron añadiendo a la lista más y más etiquetas. Actualmente, en su versión 5, el lenguaje HTML cuenta con una amplia biblioteca compuesta por 142 etiquetas semánticas, las cuales se emplean para darle formato a la estructura de las páginas web.

Estructura básica

El lenguaje HTML tiene una serie de etiquetas únicas que son empleadas para indicar las secciones o estructura del sitio web, las principales etiquetas e indipensables que no pueden faltar en ninguna página web son las siguientes:

  • <!DOCTYPE html> : Es la única etiqueta que se escribe en mayúsculas y se usa para indicarle al navegador en qué versión de HTML estamos trabajando, en este caso declara la versión 5 del lenguaje de etiquetado.
  • <html> </html> : Es la etiqueta raíz del archivo, entre estas se escribirá todo el código de nuestra página web. A esta etiqueta se le añade en atributo lang="" para especificar el idioma del contenido del sitio web.
    • <head> </head> : Dentro de esta etiqueta irá la configuración de la página web, es decir, los metadatos, los estilos CSS, el título del documento y cualquier otra información relevante para los motores de búsqueda, pero que no será visible para los usuarios.
      • <title> </title> : Etiqueta empleada para otorgarle un título a nuestra página web.
    • <body> </body>: Aquí se desarrollan la gran mayoría de las etiquetas del documento, en esta sección irá toda la información que se mostrará a los usuarios, es decir, el contenido de la página web.

Semántica

Al igual que todos los lenguajes informáticos, HTML está desarrollado en el idioma inglés, pero a diferencia de los demás lenguajes, HTML es completamente semántico, esto quiere decir que las etiquetas son muy literales, lo que significa que los nombres de las etiquetas están directamente relacionados a sus funciones y contenido.

Por ejemplo:

las etiquetas <footer> </footer>, tal como su nombre lo indica, encierran el contenido del pie de página del documento.

Las etiquetas <ul> </ul> y <ol> </ol> se usan para crear listas desordenadas y ordenadas, respectivamente. Sus nombres derivan de las siglas Unordered List y Ordered List.

Como podemos evidenciar en los ejemplos anteriores, las etiquetas nos informan sobre lo que trata su contenido, esto es la semántica

Clasificación de etiquetas

En HTML hay dos tipos de etiquetas:

  • Etiquetas compuestas: Son aquellas que requieren una etiqueta de apertura y otra de cierre para su correcta ejecución.
  • Etiquetas huérfanas: Vacías, así se les denomina a las etiquetas que no necesitan una etiqueta de cierre para su óptimo funcionamiento.

No son muchas las etiquetas vacías ya que la mayoría de las etiquetas son compuestas.

Metadatos y links

Todas las etiquetas meta, links a estilos CSS y otro tipo de links (CDN) de páginas externas se escriben dentro de la cabecera de la página web. Las etiquetas "meta" y "link" son huérfanas, es decir, que no necesitan de una etiqueta de cierre como otras etiquetas para que funcionen.

<meta charset="UTF-8"> Codificación regional de idioma que permite insertar un conjunto de caracteres epeciales y símbolos que no hay en el idioma inglés (ñ, tildes, etc).
<meta http-equiv="X-UA-Compatible" content="IE=edge"> Compatibilidad con Interent Explorer 8 y otros navegadores.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> Escala inicial del zoom de la página asociada al tamaño del ancho de pantalla de los diferentes dispositivos (diseño responsive).
<meta name="author" content="Jesús"> Define al creador de la página web.
<meta name="description" content="Documentación HTML5"> Otorga una breve descripción del contenido del sitio web.
<meta name="keywords" content="HTML, HTML5, La web, Documentación, Etiquetas HTML"> Palabras claves para que el navegador o los motores de búsqueda encuentren la página web de acuerdo a la solicitud del usuario.
<link rel="stylesheet" href="Ruta a la hoja de estilos"> Vinculación de los archivos de hojas de estilo en cascada externos para que los estilos puedan ser aplicados a los elementos de la página web.
<link rel="shortcut icon" href="Ruta a la imagen" type="image/x-icon"> Permite insertar una imagen de dimensiones pequeñas que se mostrará en la pestaña del navegador para personalizar el sitio web.

Encabezados

Las etiquetas de encabezado se usan para declarar títulos y subtítulos en el documento. Van enumeradas del 1 al 6: Este número indica la jerarquía del titular, siendo 1 el más importante y 6 el de menor relevancia.

<h1> </h1> Título principal.
<h2> </h2> Subtítulo.
<h3> </h3> Subtítulo.
<h4> </h4> Subtítulo.
<h5> </h5> Subtítulo.
<h6> </h6> Subtítulo.

Enlaces

Los enlaces nos permiten conectar páginas entre sí o secciones de una misma página para navegar entre ellas.

Hay dos tipos de enlaces:

  • URL absolutas: Links en la web.
  • URL relativas: Carpetas o archivos en nuestro ordenador.
<a href="Ruta de navegación (absoluta o relativa)"> </a> Crea un enlace a través de palabras clicables de un hipervínculo. Ejemplo: Si haces click sobre este texto serás redireccionado a otra página web.

Imágenes

<img src="Ruta a la imagen" alt="Texto alternativo"> Al hacer uso de esta etiqueta puedes insertar una imagen a tu sitio web para que sea mucho más atractivo. El atributo alt se usa para dar una breve descripción del contenido de la imagen. Puedes usar los atributos width="" y height="" para modificar las dimensiones de la imagen, ancho y alto, respectivamente.

Listas

<ol> </ol> Creación de listas ordenadas. las listas ordenas tienen algunos atributos como "reversed", que es aplicado para alterar el orden de la lista, es decir, para que empiece en orden descendente, de mayor a menor. También se puede modificar el tipo de lista y ser cambiada por letras o números romanos.
<ul> </ul> Creación de listas desordenadas.
<li> </li> Los elementos li se anidan dentro de las etiquetas ol o ul para definir qué cantidad de items tendrá la lista.
<dl> </dl> Creación de listas descriptivas, generalmente usadas para construir glosarios o diccionarios.
<dt> </dt> Define o da un título a cada término de la lista.
<dd> </dd> Define la descripción o contenido de un término.

Contenedores semánticos

<header> </header> Es el encabezado de la página web; por lo general, dentro de estas etiquetas se agrega el logo, título, lema o slogan y un menú de navegación.
<nav> </nav> Menú de navegación, es decir, sección de la página que enlaza a otra páginas web o a secciones específicas dentro de la misma página.
<article> </article> Representa un contenido completo, que está anidado dentro de las etiquetas section.
<section> </section> Divide o agrupa el contenido de la página en artículos.
<aside> </aside> Contenido que no es relevante para los usuarios.
<footer> </footer> Pie de página, aquí se indexa información de copyright y las redes sociales.
<div> </div> Divisor de contenido.
<sidebar> </sidebar> Agrupa un contenido en una columna que se dispone a la derecha de la página web.

Audio y video

<audio src="Ruta al audio" type="audio/mp3" controls> </audio> Esta etiqueta permite insertar un audio a tu página web; al usar el atributo "controls" se agregan controles para que el usuario pueda modificar el volumen, pausar o reproducir el audio.
<video src="Ruta al video" type="video/mp4" controls autoplay> </video> Esta etiqueta permite insertar un video a tu página web; al igual que con las imagenes, puedes modificar las dimensiones del video con los atributos width="" y height="". También puedes incluir el atributo "controls" para agregar controles que le permitan al usuario modificar el volumen, pausar o reproducir el video. Otro atributo de los videos es "autoplay" que hace que el video se reproduzca automáticamente cuando cargue la página.

Formularios

<form> </form> Dentro de estas etiquetas se escriben todas las etiquetas que conforman el contenido del formulario.
<fieldset> </fieldset> Crea un cuadro que encierra las entradas que se coloquen dentro de la etiqueta, en resumen, se usa para agrupar elementos relacionados entre sí.
<legend> </legend> Se coloca entre las etiquetas fieldset y su función es añadir un título al cuadro.
<label for=""> </label> La etiqueta permite dar un título o descripción a cada campo del formulario; además relaciona el texto con el campo al que corresponde.
<input> Hay varios tipos de input, pero básicamente son entradas en forma de cajas o botones donde se pide a los usuarios ingresar información.
Atributos de input
type="" Define el tipo de entrada en que se convertirá el control, puede ser un campo de texto, un campo numérico, de contraseña, etc.
name="" El valor de este campo debe coincidir con el valor del atributo for de la etiqueta label.
value="" Propiedad aplicada a los botones que permite definir el contenido que dará un concepto al botón.
placeholder="" Muestra un ejemplo de lo que el usuario debe introducir en el campo del formulario.
autocomplete="on/off" Se usa para que el input muestre sugerencias al usuario de algo que haya escrito anteriormente.
min="" max="" Sirven para poner un límite de cantidad en el campo del formulario (solo aplica para number y range).
step="" También es usando en inputs de cantidad para indicar de cuánto en cuánto saltan los números.
minlength="" maxlength="" Solo aplica para campos de texto y son usados para establecer un rango mínimo y máximo permitido de caracteres.
size="" Establece un tamaño numérico a los caracteres.
required El atributo required es empleado para indicar que un campo en un formulario es obligatorio y que el formulario no puede enviarse sin antes completar dicho campo.
checked Al usar el atributo checked logramos que una casilla de verificación o un botón de radio aparezca marcado por defecto.
autofocus El atributo autofocus se le atribuye al campo que necesitamos que aparezca seleccionado apenas cargue el formulario; generalmente se le coloca al primer campo. Este es un atributo irrepetible, es decir, solo puede colocarse una vez.
pattern="" Se emplea para limitar a que el usuario diligencie el campo siguiendo ciertos patrones o especificaciones. A las etiquetas que contengan este atributo debe añadirse también el atributo title="", el cual debe contener una descripción o ejemplo con el formato que debe ingresar el usuario para que, en caso de equivocarse, lea las indicaciones y pueda completar el campo, logrando enviar el formulario satisfactoriamente.
